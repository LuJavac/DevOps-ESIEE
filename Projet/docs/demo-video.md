# Demo Video - Checklist

Ajoutez ici le lien de votre video finale:

- Lien video: `A COMPLETER`
- Duree cible: 3 a 5 minutes

## Script recommande

1. Intro (20s)
- Nom du projet + objectif "sport proche de moi".

2. Architecture (30s)
- Frontend + Backend + PostgreSQL + Kubernetes + CI/CD.

3. Pipeline (30s)
- Montrer workflows GitHub Actions (tests, build, deploy).

4. Demo fonctionnelle (2-3 min)
- Geolocalisation utilisateur.
- Recherche par sport.
- Affichage des resultats avec distance.
- Boutons Google Maps / Waze.
- Copie adresse.

5. Validation technique (30s)
- `make test`
- `make k8s-status`

6. Conclusion (20s)
- Rappel de la couverture des consignes.
